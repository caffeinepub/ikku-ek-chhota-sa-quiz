# Ikku - Ek Chhota Sa Quiz

A romantic quiz application built with React and TypeScript.

## Publish Configuration

The deployment domain/slug is configured in `frontend/publish-domain.json`.

**Current domain:** `ikku-ek-chhota-sa-quiz`

### Domain Requirements
- Must be between 5 and 50 characters
- Can only contain lowercase letters (a-z), numbers (0-9), and hyphens (-)
- No emojis or special characters allowed

To change the publish domain, edit the `domain` field in `frontend/publish-domain.json`.
